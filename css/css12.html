display : 출력 방식

none, block, inline, inline-block

none : 출력 안됨

block : 크기지정, 위/아래 마진, 위/아래 패딩 등이 적용됨

inline : 크기지정, 위/아래 마진, 위/아래 패딩 등이 적용되지 못하지만 한 줄로 나열

inline-block : block과 inline의 특성을 모두 가짐
크기지정, 위/아래 마진, 위/아래 패딩 등이 적용되며, 한 줄로 나열 가능함

visibility : 가시성

visible, hidden

opacity : 불투명도

0~1

크기지정과 관련한 속성

실제폭 = ml+bl+pl+w+pr+br+mr

실제높이 = mt+bt+pt+h+pb+bb+mb

width:너비, height:높이

px단위, %단위, auto, vw, vh 지정 가능

margin:바깥여백, padding:안여백

px단위, %단위, auto, vw, vh 지정 가능


margin 통합 속성

margin:40px; - 상/하/좌/우 바깥여백이 모두 40px

margin:20px 50px; - 상/하 여백이 20px이고, 좌/우 여백이 50px

margin:10px 20px 30px 40px; - 상10px/우20px/하30px/좌40px

margin 세부 속성

margin-top, margin-bottom, margin-left, margin-right

만약, 상/하/좌/우의 마진을 50px로 지정하되 위쪽 여백만 10px로 할 경우
margin:50px; margin-top:10px; (O)
margin-top:10px; margin:50px; (X)
그러므로, 통합 속성을 지정 후 세부속성을 지정해야함

padding 통합 속성

padding:40px; - 상/하/좌/우 바깥여백이 모두 40px

padding:20px 50px; - 상/하 여백이 20px이고, 좌/우 여백이 50px

padding:10px 20px 30px 40px; - 상10px/우20px/하30px/좌40px

padding 세부 속성

padding-top, padding-bottom, padding-left, padding-right

만약, 상/하/좌/우의 패딩을 50px로 지정하되 위쪽 여백만 10px로 할 경우
padding:50px; padding-top:10px; (O)
padding-top:10px; padding:50px; (X)
그러므로, 통합 속성을 지정 후 세부속성을 지정해야함

border : 테두리

border 통합속성

border:두께 선모양 선색;

border 세부속성

border-width : 선두께

border-style : 선모양

border-color : 선색

border-left, border-right, border-top, border-bottom

border 세세속성

border-left-color, border-left-style, border-left-width

.....
